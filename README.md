
#  网易云播放器
## 安装依赖 以及启动命令
```
    /package.json
        npm run init-dev             安装全局&本地依赖
        npm run init-dev-g           安装全局依赖
        npm start                    启动 全部服务
        npm run start-app            启动 electron
        npm run start-api            启动 api
        npm run start-music          启动 web程序
        npm run build                打包成桌面程序

```
